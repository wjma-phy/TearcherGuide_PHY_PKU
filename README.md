# 北京大学物理学院 · 教职工Wiki系统

一个面向北京大学物理学院教职工的在线服务门户，整合学院各项业务流程与规章制度，帮助教职工快速找到需要办理的事务。

## 在线预览

> 部署后通过 HTTP/HTTPS 访问，请勿直接本地打开 `dist/index.html`（浏览器会拦截 ES 模块）。

## 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite 7
- **样式**: Tailwind CSS 3 + shadcn/ui
- **图标**: lucide-react

## 功能模块

- **入职与离职**：新教职工入职、党组织关系转接、加入工会、实验室准入、离职手续
- **教学相关**：新开课程、课程排课、课程异动、考试违纪处理、成绩录入、导师招生资格审核等
- **科研相关**：国家自然科学基金申请、横向项目合同、科研经费入账、知识产权申请、论文投稿宣传
- **人事相关**：学术假、校外兼职、职称评审、Tenure评估、请销假、师德师风评估、组建队伍
- **博士后事务**：招聘、进站、出站、证明办理、项目申报、副研究员资格评审
- **财务相关**：日常报销、大额资金使用审批、进口免税采购、家具/软件采购、耗材管理、合同签订
- **设备管理**：设备采购、设备报废、设备移交调拨
- **出国出境**：短期/长期因公出国、因公护照办理、外籍教师聘请、国际合作协议、引智项目
- **生活福利**：公费医疗报销、住院费借款、教职工体检、爱心基金、独生子女奖励、幼儿托补费、机动车通行证
- **日常行政**：用印申请、会议室预约、公共空间改造、访客预约、家属证、设施设备报修、调取监控、大屏幕使用、车队用车、临时餐卡、校外人员长期进出权限
- **实验室安全**：实验室安全管理、楼顶门禁权限
- **规章制度汇编**：按业务类别整理的规章制度
- **办公联系方式**：各办公室联系人及联系方式

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器（localhost:5173）
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
├── src/
│   ├── components/        # React 组件
│   │   ├── ContentArea.tsx    # 主内容区（首页/FAQ/规章/详情页）
│   │   ├── Sidebar.tsx        # 左侧导航栏
│   │   └── SearchBox.tsx      # 顶部搜索框
│   ├── data/              # 静态数据
│   │   └── wikiData.ts        # 所有业务数据（办事指南、FAQ、联系方式、规章制度等）
│   ├── components/ui/     # shadcn/ui 组件库
│   ├── hooks/             # 自定义 Hooks
│   ├── lib/               # 工具函数
│   ├── App.tsx            # 根布局组件
│   ├── main.tsx           # 入口文件
│   └── index.css          # 全局样式（PKU 红色主题变量）
├── dist/                  # 构建输出目录（生产部署用）
├── index.html             # HTML 入口
├── vite.config.ts         # Vite 配置
├── tailwind.config.js     # Tailwind CSS 配置
├── tsconfig.json          # TypeScript 配置
└── package.json           # 项目依赖
```

## 设计规范

- **主题色**: 北大红 `#94070A`
- **背景色**: 暖石色调 `hsl(40,20%,97%)`，减少视觉疲劳
- **设计风格**: 学术精致风，玻璃拟态卡片、时间线流程、柔和阴影
- **响应式**: 桌面端左右分栏，移动端侧边栏变为抽屉式菜单

## 部署说明

本项目为纯静态单页应用（SPA），无需后端服务。构建后 `dist/` 目录可直接部署到任意静态托管平台：

- Nginx / Apache
- GitHub Pages
- Cloudflare Pages
- Vercel / Netlify

## 数据维护

所有内容数据集中在 `src/data/wikiData.ts` 中维护，包括：

- `wikiData` — 办事指南分类及详细步骤
- `faqData` — 常见问题
- `regulationsData` — 规章制度汇编
- `contactData` — 办公联系方式
- `hotTopics` — 首页快速入口
